With controlled-width section attribute

To prevent fields from taking up too much horizontal space, use the controlled-width
attribute on rx-form-section. The fields will still stack if horizontal space is
constrained, but they will not exceed a reasonable field width. Name: Dropdown + Text Input + Dropdown Search Box: Using rx-search-box.
